# MemeGram / MemeBook

## Funktioner

- ✔ Se alla uppladdade memes (inkl. mina egna)
- ✔ Skapa konto
- ✔ Ladda upp memes
- ✔ Se enskild meme i en lightbox
- ✔ Ta bort egna memes
- ✔ Se mina uppladdade memes

### Framtida funktioner

- Like:a memes (som inloggad)
